タイトル「金のなる木」

【起動方法】
フォルダ「jade_plant」内にpdeファイルを置いてからProcessing 1.5.1で起動してください。

【GitからZIPをダウンロードした方へ】
(Windows 7以降の方)
MeiryoUIを32ptで「MeiryoUI.vlw」という名前でCreate FontしてからRunしてください。
(それ以外の方)
適当な日本語ファイルを32ptでCreate Fontしてから
PFont meiryo = loadFont("ファイル名.vlw");
に適宜書き換えてください。
